### Step 2: Deploy to Netlify

3. Drag & drop the `frontend/build` folder
4. Get your live URL!

### Step 3: Alternative - Netlify CLI
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Login
netlify login

# Deploy
cd frontend
npm run build
netlify deploy --prod --dir=build
```
